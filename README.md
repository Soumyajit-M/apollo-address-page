## 🚀 Deployment (Cloudflare Pages)

This site is deployed using **Cloudflare Pages** with auto-sync via GitHub.

### How to Deploy

1. Push `index.html` to your GitHub repository
2. Connect the repo to **Cloudflare Pages**
3. Use these build settings:

| Setting               | Value        |
|-----------------------|--------------|
| Framework preset      | None         |
| Build command         | *(empty)*    |
| Build output directory| *(empty)*    |

### Auto-Sync (How Updates Go Live)

Every time you edit and push to GitHub, Cloudflare automatically redeploys within ~30 seconds.

```
Edit index.html → git push → Cloudflare detects change → Live ✅
```

## ✏️ How to Edit the Website

### Option 1 — Directly on GitHub (Easiest)
1. Open your repo on GitHub
2. Click `index.html` → click the ✏️ pencil icon
3. Make your changes
4. Click **"Commit changes"**
5. Site updates automatically ✅

### Option 2 — On Your Computer
```bash
git clone https://github.com/YOUR_USERNAME/fixnova
# Edit index.html in VS Code or any editor
git add .
git commit -m "your update message"
git push
```
